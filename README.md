# mirai-console-jcr-plugin
## J Cpp Reference Plugin

在QQ群内搜索 [CppReference](https://en.cppreference.com/w/) 的文档

# 用法
```shell
# 消息检测
c <keyword>   # 搜索C文档
cpp <keyword> # 搜索CPP文档

# 控制台命令
/jcr update   # 更新CPP索引
```

# 索引来源
c 索引来自官网离线文档，手动解析生成
cpp 索引来自 https://github.com/Guyutongxue/cppreference-index 感谢 @Guyutongxue
